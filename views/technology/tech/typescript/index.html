# 基础

# 什么是Javascript

JavaScript 是一种轻量级的跨平台 解释性脚本语言。

特点：

  • 无需编译，只需嵌入HTML代码中，就能由浏览器逐行加载解释执行
  • 跨平台与操作系统无关
  • 事件驱动
  • 变量为弱类型
  • 没有类的概念

# 什么是TypeScript?

TypeScript是 Microsoft 开发和维护的一种面向对象的编程语言，它是JavaScript的一个超集，扩展了JavaScript的语法。

特点：

  • TypeScript 是完全兼容 JavaScript 的，它不会修改 JavaScript 运行时的特性。
  • TypeScript 可以编译为 JavaScript，运行在任何能运行 JavaScript 的环境中。
  • TypeScript 拥有很多编译选项，类型检查的严格程度由你决定。
  • TypeScript 增强了编辑器（IDE）的功能，提供了代码补全、接口提示、跳转到定义、代码重构等能力。

# TS与JS 对比

TypeScript 可以使用 JavaScript 中的所有代码和编码概念是JavaScript的一个超集。
